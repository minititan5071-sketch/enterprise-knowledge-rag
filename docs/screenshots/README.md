# Screenshot Checklist

Capture these screenshots after starting the demo stack and loading the synthetic fintech documents.

Recommended filenames:

- `01_login.png`: Streamlit login and registration page.
- `02_workspace.png`: Workspace selection and member management.
- `03_document_upload.png`: Document upload page showing completed ingestion.
- `04_rag_answer_with_citations.png`: Ask page with an answer, source documents, snippets, and confidence score.
- `05_audit_logs.png`: Query History page showing audit records.
- `06_evaluation_dashboard.png`: Evaluation Results page after running the fintech eval set.

Guidance:

- Use the `HarbourBridge Knowledge Base` workspace.
- Ask a question from `docs/demo_script.md` so citations are easy to explain.
- Avoid showing real environment variables, tokens, browser password managers, or local filesystem paths outside the project.
- Keep screenshots focused on the product surface, not the terminal.
